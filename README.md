# re-cnode

🐤 Just another implementation of cnodejs.org

![screenshot](/public/re-cnode.png)

## Quick start

```bash
# clone repository
$ git clone https://github.com/bushuai/re-cnode.git

# start server
$ cd server && yarn start

# start client(root directory)
$ yarn start
```

## Dependencies

### client side

- react
- redux
- axios
- react-redux
- styled-components
- react-thunk
- react-router-dom
- ...

### server side

- cherrio
- superagent
- nodemon
